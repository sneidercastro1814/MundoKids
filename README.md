# 🦉 MundoKids - ¡Aprende Jugando!

App educativa gamificada estilo Duolingo para niños.

## 🎮 Características

- **6 Materias**: Matemáticas, Inglés, Ciencias, Historia, Religión, Física
- **360 Preguntas**: 20 preguntas × 3 niveles × 6 materias
- **Gamificación estilo Duolingo**:
  - ❤️ Sistema de 5 vidas/corazones
  - ⚡ Puntos XP con bonus por racha
  - 🔥 Rachas de aciertos consecutivos
  - ⭐ Estrellas de rendimiento (1-3)
  - 🎉 Confeti al acertar
  - 🦊🦉🐸🦁🕊️🐱 Mascotas animadas por materia
  - 🔒 Niveles desbloqueables (Fácil → Medio → Difícil)
  - 🏆 Sistema de logros (6 badges)
  - 👤 Perfil con estadísticas y progreso

## 🚀 Instalación

```bash
# 1. Instalar dependencias
npm install

# 2. Iniciar en modo desarrollo
npm run dev

# 3. Abrir en el navegador
# http://localhost:3000
```

## 📦 Construir para producción

```bash
npm run build
```

Los archivos se generan en la carpeta `dist/`.

## 🛠️ Tecnologías

- React 18
- Vite 5
- CSS Animations
- Google Fonts (Fredoka One + Nunito)

## 📱 Responsive

La app funciona en móvil, tablet y escritorio.
